
## Passo-a-passo para execução

1. Clonar o repositório
2. Instalar as dependências
   `
   npm install
   `
3. Executar a aplicação usando o npm ou yarn:
   `
   npm start
   `
   ou
   `
   yarn start
`

## Tecnologias utilizadas

[ReactJS](https://reactjs.org/)  
[Material UI](https://material-ui.com/)
